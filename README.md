# SwingBy

[![Backend CI](https://github.com/4alkubati/SWINGBY/actions/workflows/backend.yml/badge.svg)](https://github.com/4alkubati/SWINGBY/actions/workflows/backend.yml)
[![Mobile CI](https://github.com/4alkubati/SWINGBY/actions/workflows/mobile.yml/badge.svg)](https://github.com/4alkubati/SWINGBY/actions/workflows/mobile.yml)

**SwingBy** is a dual-sided service marketplace connecting local service businesses with people who need them — Uber meets Thumbtack, built for Calgary and expanding across North America.

- **Clients** post a job (or browse nearby businesses on a map), review bids, and book.
- **Businesses** discover posted jobs, express interest with a quote, assign employees, and get paid through escrow.

🌐 [swingbyy.com](https://swingbyy.com)

## How it works

Client posts a job → businesses express interest with quotes → client accepts one → booking created → employee assigned → work done with photo proof + live status timeline → escrowed payment released.

## Quickstart

**Backend** (from `backend/`, needs a `.env` — see `docs/RUNNING_LOCALLY.md`):

```bash
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**Mobile** (from `mobile/`):

```bash
npx expo start --clear
```

**Tests + lint** (what CI runs):

```bash
ruff check backend/ && black --check backend/ && pytest backend/tests -q
```
