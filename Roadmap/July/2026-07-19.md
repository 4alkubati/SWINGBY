## 📖 Glossary
- **HIBP** — HaveIBeenPwned. Supabase checks new passwords against its breach corpus and rejects known-leaked ones.
- **Advisor** — Supabase's built-in security/performance linter. "0 open advisors" is a launch-checklist line item.
- **Migration** — a SQL file that changes database structure. Filed in `docs/`, applied by hand in the SQL editor.
- **EAS** — Expo Application Services; the cloud builder that produces the real installable app.
- **Expo Go** — the sandbox app for quick previews. Uses its own native config, so native keys (like Maps) don't apply.
- **SHA-1 fingerprint** — the signature of the key your Android app is signed with; Google uses it to prove a Maps request came from your app.
