## How to use

### Paste into Notion content queue
1. Open your Notion content calendar database.
2. Copy a post from the relevant file.
3. Paste into a new row. Set the Platform, Date, and Status fields.
4. Edit the [brackets] and any placeholder text before scheduling.

### Bulk scheduling (Buffer / Later / Hootsuite)
1. Export the file content into a spreadsheet: one row per post (date, platform, text, image note).
2. Import the CSV into your scheduler.
3. Attach images separately — see image notes in each post.

### Before you post
- Replace all `[your handle]`, `[link in bio]`, and bracket placeholders.
- Add your chosen image or video (see image note per post).
- Hashtags are pre-filled but audit them against current trends before posting.
- For IG: add Stories on the same day to boost post reach.

## Brand voice quick-check

Before posting, ask: does this sound like SwingBy?

✅ Direct, warm, specific — "Book a vetted cleaner in Beltline today."  
✅ Pro-small-business, pro-customer, never anti-anything.  
❌ No: "Revolutionize," "unlock," "game-changing," "synergy."  
❌ No: strings of emoji (one is fine, three is too many).  
❌ No: all-caps in body copy, vague CTAs ("click here").  

Full guidelines: [../09-brand-guidelines.md](../09-brand-guidelines.md)
